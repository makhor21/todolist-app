# Todo List App with MySQL Database

### Application Features

```bash
1. Authroize Users
2. Add a Todo
3. Update a Todo to has Done
4. Delete a Todo
5. Pagination
```

### For Using This Application Run These Commands

```bash
1. npm install
2. run MySQL Services 
2. npm start
3. Open Your Browser
4. Search URL http://localhost:8080
```
